snowflakify


A complete Snowflake ID generator in TypeScript.
Generate, manage, create custom snowflakes, and more...


About

Snowflakify is a complete Node.js module for snowflake ID generation written in TypeScript.

  • IDs based on Workers/Cluster, internal IPv4/MAC addresses
  • Snowflake destructuring
  • BigInt based
  • Custom epoch
  • Custom snowflake fragments
    with customizable bit length:
    • Timestamp
    • Random
    • Worker ID
    • Process ID
    • IPv4 Address
    • MAC Address
    • Sequence
